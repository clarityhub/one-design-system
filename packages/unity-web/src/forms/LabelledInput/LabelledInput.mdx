# LabelledInput

The `LabelledInput` component is a controlled input. You <b>must</b> provide a
`value` and `onChange`.

## \<LabelledInput /\> component

```js
import LabelledInput from '@clarityhub/unity-web/lib/forms/LabelledInput';
```

### Controlled
<Playground>
    {() => {
        class Example extends Component {
            constructor(props) {
                super(props);
                this.state = {
                    value: '',
                };
            }

            render() {
                return (
                    <div>
                        <InputGroup>
                            <LabelledInput
                                prefixIcon={<SearchIcon />}
                                label="Name"
                                onChange={e => this.setState({ value: e.target.value })}
                                value={this.state.value}
                            />
                        </InputGroup>
                        <InputGroup>
                            <LabelledInput
                                label="Name"
                                placeholder="Hello World"
                                onChange={e => this.setState({ value: e.target.value })}
                                value={this.state.value}
                            />
                        </InputGroup>
                    </div>
                )
            }
        };

        return <Example />;
    }}
</Playground>
