# LabelledTextarea

The `LabelledTextarea` component is a controlled input. You <b>must</b> provide a
`value` and `onChange`.

## \<LabelledTextarea /\> component

```js
import LabelledTextarea from '@clarityhub/unity-web/lib/forms/LabelledTextarea';
```

### Controlled

<Playground>
    {() => {
        class Example extends Component {
            constructor(props) {
                super(props);
                this.state = {
                    value: '',
                };
            }

            render() {
                return (
                    <div>
                        <InputGroup>
                            <LabelledTextarea
                                label="Name"
                                onChange={e => this.setState({ value: e.target.value })}
                                value={this.state.value}
                            />
                        </InputGroup>
                        <InputGroup>
                            <LabelledTextarea
                                label="Name"
                                placeholder="Hello World"
                                onChange={e => this.setState({ value: e.target.value })}
                                value={this.state.value}
                            />
                        </InputGroup>
                    </div>
                )
            }
        };

        return <Example />;
    }}
</Playground>
