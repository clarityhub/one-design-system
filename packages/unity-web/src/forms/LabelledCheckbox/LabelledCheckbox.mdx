# LabelledCheckbox

The `LabelledCheckbox` component is a controlled input. You <b>must</b> provide a
`selected` and `onChange`.

## \<LabelledCheckbox /\> component

```js
import LabelledCheckbox from '@clarityhub/unity-web/lib/forms/LabelledCheckbox';
```
