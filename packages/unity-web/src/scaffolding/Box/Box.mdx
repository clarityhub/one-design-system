# Box

Box is a primitive that you can use to space out elements on a page.

## \<Box /\> component

```js
import Box from '@clarityhub/unity-web/lib/scaffolding/Box';
```

## Examples

### Colored Boxes

Boxes can be colors using `background`, `color`, or `type`:

<Playground>
	<Box align="center" background="primary">
		primary
	</Box>
	<Box align="center" type="danger">
		danger (type)
	</Box>
	<Box align="center" color="danger">
		danger (color)
	</Box>
</Playground>

### Boxes as rows

<Playground>
	<Box
		direction="row"
		padding="medium"
		style={{ border: '10px solid blue', width: '100px' }}
	>
		<Box margin={{ top: "small" }} padding="small" style={{ background: 'lightgrey' }} />
		<Box padding="medium" style={{ background: 'gray' }} />
	</Box>
</Playground>

### Various Padding and Margins

<Playground>
	<Box padding="large" style={{ background: 'blue' }}>
		<div style={{ backgroundColor: 'white' }}>padding</div>
	</Box>

	<div style={{ background: 'red' }}>
		<Box margin="large" padding="large" style={{ background: 'blue' }}>
			<div style={{ backgroundColor: 'white' }}>padding + margin</div>
		</Box>
	</div>
</Playground>
