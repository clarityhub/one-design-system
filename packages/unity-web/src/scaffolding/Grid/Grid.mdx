# Grid

Show a 2 column grid that changes to 1 column at mobile size. Useful for 
displaying a list of Cards.

## Grid versus FlexGrid

FlexGrid is useful when you need to layout a set of visuals and you don't care
about the size of the device.

You should use Grid when you need to display a list of item data and you want
to display the data in Cards (or other similar components).

## \<Grid /\> Component

```js
import { GridLayout, GridItem } from '@clarityhub/unity-web/lib/scaffolding/Grid';
```
