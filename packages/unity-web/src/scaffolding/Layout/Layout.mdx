# Layout

The Layout primitive is a helpful scaffolding component that takes up the fullscreen.

It will take up `100vh` and `100%` width.

## \<Layout /\> component

```js
import Layout from '@clarityhub/unity-web/lib/scaffolding/Layout';
```
