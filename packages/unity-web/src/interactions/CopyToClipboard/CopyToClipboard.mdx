# CopyToClipboard

Use the CopyToClipboard interaction to allow users to quickly copy text to their clipboard

```js
import CopyToClipboard from '@clarityhub/unity-web/lib/interactions/CopyToClipboard';
```

Make sure you also have `ToastManagerProvider` as a wrapper.

## CopyToClipboard Example

<Playground>
    <ToastManagerProvider>
        <CopyToClipboard>
			{({ onClick }) => (
				<InputGroup>
					<LabelledInput
						label="Access Key ID"
                        readonly
						value={"123456"}
					/>
					<InputGroupAppend>
						<Button
                            outline
                            type="primary"
                            onClick={() => onClick("123456")}
                        >
                            Copy
                        </Button>
					</InputGroupAppend>
				</InputGroup>
			)}
		</CopyToClipboard>
    </ToastManagerProvider>
</Playground>
