# ToastManager

Wrap your app with the provider:

```js
import { ToastManagerProvider } from '@clarityhub/unity-web/lib/contexts/ToastManager';
```

To use toasts in your app, use the following with `useContext`:

```js
import ToastManager from '@clarityhub/unity-web/lib/contexts/ToastManager';
```

## ToastManager Usage Example

```js
import { useContext } from 'react';
import ToastManager from '@clarityhub/unity-web/lib/contexts/ToastManager';

const CopyToClipboard = ({ children }) => {
	const { createToast } = useContext(ToastManager);

	const onClick = (data) => {
		copyToClipboard(data);

		createToast({
			dedupId: 'copy-to-clipboard',
			message: 'Copied!',
			type: 'success',
		});
	};

	return children({ onClick });
};

```
