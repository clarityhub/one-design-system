# Breadcrumbs

Use breadcrumbs to show a user where they are in the sitemap.

```js
import Breadcrumbs from '@clarityhub/unity-web/lib/components/Breadcrumbs';
```

## Example

<Playground>
    <Breadcrumbs
        crumbs={[
            {
                title: 'Home'
            }
        ]}
    />
    <Breadcrumbs
        crumbs={[
            {
                title: 'Home',
                path: '/',
            },
            {
                title: 'Admin',
            },
        ]}
    />
    <Breadcrumbs
        crumbs={[
            {
                title: 'Home',
                path: '/',
            },
            {
                title: 'Admin',
                path: '/admin',
            },
            {
                title: 'Users',
                // no path
            }
        ]}
    />
</Playground>

## Link Renderer

By default, the breadcrumbs will use a Link to render the given path.

You can override how breadcrumbs are rendered (so you can use React Router Links for example) by providing a `linkRenderer`.

The default renderer looks like:

```js
const defaultLinkRenderer = ({ path, children }) => (
	<Link href={path}>{children}</Link>
);
```
