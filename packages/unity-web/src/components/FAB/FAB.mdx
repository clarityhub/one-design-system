# FAB

A FAB (Floating Action Button) is special type of button that is a circle that
can be positioned at the bottom right of a page.

```js
import FAB from '@clarityhub/unity-web/lib/components/FAB';
```

## Prop Types

A FAB takes the same props as a [Button](#/packages-unity-web-src-components-button-buttons#prop-types).
