# ButtonGroups

A `ButtonGroup` groups a set of actions together that act like checkboxes or
radio inputs. For example, displaying days of the week would be better suited
as a ButtonGroup then as distinct checkboxes. 

```js
import { ButtonGroup, ButtonGroupItem } from '@clarityhub/unity-web/lib/components/Buttons';
```

## Examples

<Playground>
    <ButtonGroup>
        <ButtonGroupItem>Mon</ButtonGroupItem>
        <ButtonGroupItem selected>Tue</ButtonGroupItem>
        <ButtonGroupItem>Wed</ButtonGroupItem>
        <ButtonGroupItem>Thu</ButtonGroupItem>
        <ButtonGroupItem selected>Fri</ButtonGroupItem>
        <ButtonGroupItem>Sat</ButtonGroupItem>
        <ButtonGroupItem>Sun</ButtonGroupItem>
    </ButtonGroup>
</Playground>
