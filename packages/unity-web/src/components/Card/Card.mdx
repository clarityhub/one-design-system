# Card

Cards provide a content container that you can use to display grouped content.

```js
import Card, {
    CardHeader,
    CardFooter,
    CardImage,
    CardBody,
    CardActions,
} from '@clarityhub/unity-web/lib/components/Card';
```

## Simple Card

In all examples, the headings and text use the `Typography` components.

<Playground>
    <Card>
        <CardBody>
            <Typography type="text">
                Default
            </Typography>
        </CardBody>
    </Card>
    <Card type="notification" style={{ marginTop: '1rem'}}>
        <CardBody>
            <Typography type="text" color="white">
                Notification
            </Typography>
        </CardBody>
    </Card>
</Playground>
