# Menu

Show a pop-up menu in navigation or when there are many options
a user can select.

Menus will always appear fixed to the bottom-left of the child
element, but will fix to the bottom-right if the window width
would be less than the width of the popover.

## \<Menu /\> component

```js
import { Menu, MenuItem, MenuItemDivider } from '@clarityhub/unity-web/lib/components/Menu';
```
