# Link

The Link component is used for displaying internal or external links. There are a few variants of links.
By default, links will be displayed inline. To display a link as a block, use the type `blockLink`. To use your own
component for rendering links, pass in a component to the `component` prop.


```js
import Link, { LinkButton } from '@clarityhub/unity-web/lib/components/Link';
```

## LinkButton Examples

You can use a Link as a Button by using the `LinkButton` component and passing
through the `history` prop.

```jsx
import React from 'react';
import { withRouter } from 'react-router';
import { LinkButton } from '@clarityhub/unity-web/lib/components/Link';

const Wrapper = ({ history }) => (
    <LinkButton type="danger" history={history} to="my/href">
        Danger
    </LinkButton>
);

export default withRouter(Wrapper);
```
