# ButtonGroup

A `ButtonGroup` groups a set of actions together that act like checkboxes or
radio inputs. For example, displaying days of the week would be better suited
as a ButtonGroup then as distinct checkboxes. 

```js
import ButtonGroup from '@clarityhub/unity-web/lib/components/ButtonGroup';
```
