# Lists

There are two types of lists available - ordered and unordered.

## Components

```js
import { OrderedList, UnorderedList, ListItem} from '@clarityhub/unity-web/lib/components/List';
```

## Examples

### Ordered List

<Playground>
    <OrderedList>
        <ListItem>First item</ListItem>
        <ListItem>Second item</ListItem>
        <ListItem>Third item</ListItem>
    </OrderedList>
</Playground>

### Unordered list

<Playground>
    <UnorderedList>
