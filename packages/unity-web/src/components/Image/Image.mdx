# Image

This Image component is used for displaying images.

```js
import Image from '@clarityhub/unity-web/lib/components/Image';
```

## Examples

<Playground>
    <Image style={{ width: '20rem' }} src="/public/get-started.png" alt="Get Started" />
    <Image style={{ width: '20rem' }} withSpacing src="/public/get-started.png" alt="Get Started" />
    <Image style={{ width: '20rem' }} center src="/public/get-started.png" alt="Get Started" />
    <Image style={{ width: '20rem' }} center withSpacing src="/public/get-started.png" alt="Get Started" />
</Playground>


Image fails to load:

    <Image
        src="/public/invalid-url.png"
