# JSONAPITable

The JSONAPITable component allows you to create tables from JSON API style data.

## \<JSONAPITable /\> component

```js
import { JSONAPITable } from '@clarityhub/unity-web/lib/components/Table';
```

## Simple Example

<Playground>
    <JSONAPITable
        columns={[
            ['Name', 'name'],
            ['Description', 'description'],
        ]}
        data={{
            data: [{
                name: 'Clarity Hub', 
                description: 'Wow',
            }, {
                name: 'Mantaray AR',
                description: 'AR',
            }]
        }}
    />
</Playground>
