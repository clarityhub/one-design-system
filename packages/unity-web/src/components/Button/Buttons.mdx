# Buttons

Buttons are used for triggering various actions.

```js
import Button from '@clarityhub/unity-web/lib/components/Button';
```

## Button Types

There are 4 main button types: default, white, primary, and danger.

* **Default buttons** should be used for actions that will not cause any changes to happen. For example, a Cancel button may close a modal, but it will not cause any changes to happen.
* **White buttons** are used when you have a non-white background color and you need to use a default button that may have variants. A white button should normally be used the same way a default button would be used.
* **Primary buttons** are for the primary action you would like a user to take. These buttons should not have any destructive consequence to clicking them.
* **Danger buttons** are for actions that would cause destructive consequences to occur; like deleting an item or unscheduling a campaign.

Otherwise, you can have buttons become anchor tags.

## Sizes

You can change the size of buttons for use in different contexts.

* Small buttons may be used in table views
* The default size of buttons should be used in most cases
* Large buttons can be used for calls to action

* Outline buttons should only be used for secondary actions
* Text buttons can be used to de-emphasis an action
