## Examples

### Text versus Paragraph

By default, if no `type` is provided to `Typography`, then `text` is used.

If you want to have multiple paragraphs of readable text, use `type="p"`.

<Playground>
    <Typography>Lorem ipsum dolor sit amet, <i>consectetur adipiscing elit</i>. Donec odio ipsum, <strong>sodales et ligula sed</strong>, venenatis cursus libero. Suspendisse sit amet tempus tortor, tincidunt egestas tortor. In hac habitasse platea dictumst. Sed consectetur nulla eget diam tincidunt eleifend.</Typography>
    <Typography type="p">Lorem ipsum dolor sit amet, <i>consectetur adipiscing elit</i>. Donec odio ipsum, <strong>sodales et ligula sed</strong>, venenatis cursus libero. Suspendisse sit amet tempus tortor, tincidunt egestas tortor. In hac habitasse platea dictumst. Sed consectetur nulla eget diam tincidunt eleifend.</Typography>
</Playground>
