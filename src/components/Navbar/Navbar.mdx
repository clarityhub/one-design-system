# Navbar

The `Navbar` (aka Header or App Bar) is the header of the app.

## \<Navbar /\> component

```js
import Navbar, { NavItem } from '@clarityhub/one/components/Navbar';
```

## Prop Types

<PropsTable of={Navbar} />

## Links

The `Link` component used is agnostic of any router that you may use in your project.

Because of this, you will need to supply a `profileMenuItemRenderer` and `brandRenderer`
that returns the `react-router-dom` or `@reach/router` link you may want to use.
