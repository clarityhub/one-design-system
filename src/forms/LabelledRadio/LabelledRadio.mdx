# LabelledRadio

The `LabelledRadio` component is a controlled input. You <b>must</b> provide a
`selected` and `onChange`.

## \<LabelledRadio /\> component

```js
import LabelledRadio from '@clarityhub/one/forms/LabelledRadio';
```
