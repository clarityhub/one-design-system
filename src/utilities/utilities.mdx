# Utilities

## Color Utilities

The color utilities provide functions to manipulate colors.

```js
import { darken, lighten, desaturate } from '@clarityhub/one/utilities/color';

console.log(darken(colors.primary, 0.3 /* defaults to 0.2 */));
console.log(lighten(colors.primary, 0.3 /* defaults to 0.2 */));
console.log(desaturate(colors.primary, 0.3 /* defaults to 0.2 */));
```

Color transformations are cached to help improve speed when used in components.
